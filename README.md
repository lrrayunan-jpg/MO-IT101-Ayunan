MotorPH Dashboard

MotorPH is a web-based motorcycle rental and management system. It provides role-based access for admins and customers, with a premium, interactive UI for browsing, booking, and managing motorcycles.

Features
Customer

View a gallery of available motorcycles with images and prices.

Click a motorcycle to auto-select it in the booking form.

Fill in name, bike, and date to book a motorcycle.

Logout functionality.

Admin

View the full inventory of motorcycles with prices.

View all customer bookings.

Role-based access ensures admins cannot book motorcycles.

Logout functionality.

General

Responsive and premium UI using Orbitron font, modern card layouts, and hover effects.

Future Improvements

Store bookings in a database instead of localStorage.

Add payment integration for bookings.

Add search and filter functionality in the gallery.

Enhance admin inventory management (add/edit/delete motorcycles).
Role-based UI rendering: Customers see gallery and booking form; admins see inventory and bookings only.

Session handling with localStorage for login persistence.
